### 🧟 角色概览

> **亡灵契约师（Necrobinder）** 是《杀戮尖塔2》中的全新职业，以"代理作战"的独特方式战斗，依靠名为 **奥斯提（Osty）** 的巨型骷髅手来进行攻击与防御。

| 属性 | 数值 |
|------|------|
| 初始生命值 | **66 HP**（全角色最低） |
| 初始能量 | 3 |
| 难度定位 | ⭐⭐⭐⭐⭐ 高复杂度 |
| 解锁条件 | 使用执政官（Regent）通关一次 |
| 初始遗物 | **被束缚的命匣（Bound Phylactery）** |

> 💡 **角色定位**：脆弱但战术密集的法师型角色，需要同时管理多个资源与机制，上限极高，下限极低。

---

### ⚙️ 四大核心机制

#### 1. 🖐️ 奥斯提（Osty）—— 骷髅手系统

奥斯提是亡灵契约师的标志性机制，一只独立行动的巨型骷髅手。

```
核心特性：
✦ 独立攻击：每回合自动对敌人造成伤害，不受玩家减益影响
✦ 伤害吸收：奥斯提存活时，替玩家承受所有伤害
✦ 格挡共享：玩家与奥斯提共享格挡值
✦ 召唤成长：通过"召唤"效果复活或提升其最大生命值
✦ 视觉成长：奥斯提HP越高，模型越大
```

> ⚠️ **重要**：奥斯提的伤害**不吃**玩家的力量加成，提升其伤害需要通过专属卡牌。

#### 2. ☠️ 灾厄（Doom）—— 延迟斩杀机制

灾厄是亡灵契约师最独特的机制——一种**延迟处决**系统。

```
核心规则：
✦ 灾厄是一个施加在敌人身上的 debuff，不随时间自动减少
✦ 当敌人身上的灾厄层数 ≥ 其当前生命值时，该敌人立即被处决
✦ 处决无视格挡和反伤
✦ 部分卡牌会对玩家自身施加厄运，形成高风险对攻局面
```

#### 3. 👻 灵魂（Souls）—— 过牌与循环资源

灵魂是亡灵契约师维持运转节奏的关键资源。

```
核心规则：
✦ 灵魂令牌：0费技能牌，效果为"抽2张牌，消耗"
✦ 打出后立即离场，不污染牌库
✦ 通过各种卡牌将灵魂加入抽牌堆、手牌或弃牌堆
✦ 是实现"无限循环"的核心引擎
```

#### 4. ✨ 虚无（Ethereal）—— 自动精简机制

亡灵契约师可以临时为卡牌添加"虚无"词条，实现自动消耗和超模数值。

```
虚无规则：
✦ 虚无牌在回合结束时若仍在手牌，会被直接消耗（烧掉）
✦ 虚无牌普遍具有远超同费卡的高数值
✦ 自动消耗 = 牌库自动精简 = 循环越来越快
```

### 🎯 五大流派构筑

#### 🌀 流派一：虚无永动流（最主流）

> **核心思路**：通过虚无机制实现自动精简，配合亡魂牵引的叠加伤害达成无限循环。

**必要组件：**

```
过牌引擎：
  • 领会（Enlighten）★★★★★ —— 虚无+抽4张
  • 书页风暴（Pagestorm）★★★★★ —— 抽到虚无牌时额外抽1张
  
虚无核心：
  • 亡魂牵引 ★★★★★ —— 每打1张虚无+5伤
  • 刺破帷幕 ★★★★★ —— 免费打出虚无牌
  • 女妖之嚎 ★★★★★ —— 虚无减费AOE
  • 坟冢爆射（升级）★★★★★ —— 去除消耗，重复检索
  
防御核心：
  • 灰烬之灵 ★★★★★ —— 虚无+大量格挡
  • 违逆 ★★★★★ —— 虚无+格挡
  
循环辅助：
  • 领域 ★★★★★ —— 每回合+1费
  • 降灵（升级0费）★★★★★ —— 灵魂生成
  • 预借时间 ★★★★★ —— 爆费核心
```

**推荐卡组（约18-22张）：**

```
输出核心：
  ×1 亡魂牵引
  ×1 刺破帷幕
  ×1-2 女妖之嚎
  ×1 坟冢爆射（升级）

防御核心：
  ×1-2 灰烬之灵
  ×2-3 违逆
  ×1-2 守墓人

过牌引擎：
  ×1-2 领会（升级优先）
  ×1 书页风暴
  ×1 降灵（升级后0费）
  ×1 领域

辅助：
  ×1-2 恐惧（施加易伤）
  ×1 鬼火（0费+1费，消耗）
  ×1 预借时间（若走爆费路线）
```

**两种运转模式：**

```
模式A：预借时间永动（爆费循环）
  1. 用雕琢打击为领会附加"虚无"
  2. 用预借时间(0费) → 获得2点费用，触发虚无计数
  3. 用领会(1费) → 虚无 + 抽4张，继续循环
  4. 重复2-3步骤，叠加亡魂牵引伤害
  5. 打出亡魂牵引造成致命伤害
  
  ⚠️ 预借时间会叠加灾厄，建议配合"灾厄护符"遗物

模式B：书页风暴+灰烬之灵（攻防一体）
  1. 用领会快速过滤牌库，找到关键牌
  2. 打出书页风暴后，每抽到虚无牌额外抽1张
  3. 打出灰烬之灵 → 获得格挡 + 触发书页风暴再抽1张
  4. 用刺破帷幕免费打出女妖之嚎或积累亡魂牵引
  
  ⚠️ 极度依赖费用！需要用领域+鬼火维持费用
```

**升级优先级：**

```
第一优先：
  ✦ 灰烬之灵 → 格挡数值大幅提升
  ✦ 领会 → 抽牌数4→5，循环效率飞跃
  ✦ 亡魂牵引 → 基础伤害提升

第二优先：
  ✦ 降灵 → 0费，极大提升灵活性
  ✦ 坟冢爆射 → 去除消耗，可重复检索
  ✦ 预借时间 → 回复3点费用

第三优先：
  ✦ 书页风暴 → 触发条件更宽松
  ✦ 领域 → 配合刺破帷幕更稳定
```

#### 👻 流派二：灵魂抽牌流（挽歌无限流）

> **核心思路**：以"灵魂"机制为核心，通过挽歌大量生成灵魂牌，配合重构的重放效果实现无限抽牌循环，再将过牌转化为伤害。

> ⚠️ **注意**：该流派容易爆牌（手牌上限10张），需在8张手牌时使用灵魂卡。使用"唤起"获得的能量下回合才完全生效，需计算好防御！

**必要组件：**

```
核心引擎：
  • 挽歌 ★★★★★ —— 消耗能量获得灵魂+召唤，流派绝对核心
  • 重构 ★★★★★ —— 升级后0费，给挽歌附加"重放"，实现一回合多次施放
  • 唤起 ★★★★★ —— 1费，有灵魂时获得2-3能量，核心能量来源

灵魂生成/过牌：
  • 守墓人 ★★★★★ —— 1费格挡+注入灵魂，前期过渡+后期运转
  • 切断 ★★★★★ —— 1费造成伤害+生成灵魂，伤害高自带过牌
  • 捕捉灵魂 ★★★★ —— 0费过牌，费用低适合快速循环
  • 一瞥彼岸 ★★★★ —— 1费消耗1能量，将3张灵魂加入手牌
  • 降灵 ★★★★ —— 战斗中烧牌+生成灵魂，精简+循环

输出终端：
  • 纠缠 ★★★★★ —— 每抽1张牌对随机敌人造成3伤（升级4伤），灵魂流最佳搭档
  • 死亡行军 ★★★★ —— 高费爆发，适合斩杀
  • 亡灵精通 ★★★★ —— 防御转群体输出，备选输出方案
  • 出击 ★★★★ —— 核心输出手段，等效"肚皮"

能量补充：
  • 鬼火 ★★★★ —— 升级后保留，存起来配合挽歌爆发
  • 预借时间 ★★★★ —— 0费能量+过牌，启动回合使用
  • 领域 ★★★★ —— 每回合+1费，永动辅助
```

**推荐卡组（约18-22张）：**

```
核心引擎：
  ×1 挽歌（必拿）
  ×1 重构（必拿，升级优先）
  ×1 唤起

灵魂生成/过牌：
  ×1-2 守墓人（必拿）
  ×1-2 切断
  ×1 捕捉灵魂
  ×1 降灵（升级优先）
  ×0-1 一瞥彼岸

输出终端：
  ×1-2 纠缠（必拿）
  ×0-1 死亡行军
  ×1 出击
  ×0-1 亡灵精通

能量补充：
  ×1 鬼火（升级优先）
  ×0-1 预借时间
  ×1 领域
```

**两种运转模式：**

```
模式A：重构挽歌无限循环（标准型）
  1. 用预借时间(0费)或保留的鬼火获得初始能量
  2. 用重构(0费)给挽歌附加"重放"
  3. 打出挽歌，消耗所有能量获得大量灵魂+召唤
  4. 用灵魂牌不断抽牌，触发纠缠的抽牌伤害
  5. 循环反复，纠缠伤害指数级增长
  
  ⚠️ 重构升级为0费是COMBO流畅运转的基石，必须优先升级

模式B：预借时间灵魂循环（轻量型）
  1. 用预借时间(0费)获得能量+自身获得灾厄
  2. 用切断/捕捉灵魂消耗能量生成灵魂牌
  3. 灵魂牌抽回预借时间，形成小循环
  4. 每回合稳定过牌+输出，纠缠持续成长
  
  ⚠️ 预借时间会叠加灾厄，建议配合"灾厄护符"遗物

**升级优先级：**

```
第一优先：
  ✦ 重构 → 0费是整套COMBO流畅运转的基石
  ✦ 挽歌 → 获得更多灵魂，直接提升效率
  ✦ 纠缠 → 每抽牌伤害从3→4，输出大幅提升

第二优先：
  ✦ 鬼火 → 获得保留特性，启动灵活性飞跃
  ✦ 唤起 → 获得的能量更多
  ✦ 守墓人 → 格挡数值提升

第三优先：
  ✦ 降灵 → 0费，精简更灵活
  ✦ 切断 → 伤害+灵魂生成量提升
  ✦ 领域 → 配合更多卡牌
```

#### 💀 流派三：灾厄爆发流

> **核心思路**：利用灾厄机制进行延迟斩杀，搭配高数值直伤解决前期压力。灾厄层数 ≥ 敌人当前HP时直接处决，无视格挡与反伤。

> ⚠️ **注意**：纯灾厄流前期清怪慢，**必须搭配直伤卡牌过渡**！灾厄默认在敌人行动回合结束后结算，需注意叠加护甲或使用"末日降临"提前结算。

**必要组件：**

```
灾厄施加核心：
  • 无处可逃 ★★★★★ —— 15灾厄，逐次成长（15→20→30），BOSS杀手
  • 瘟疫携带者 ★★★★★ —— 奥斯提每次攻击自动附带5灾厄，持续叠层
  • 负脉冲 ★★★★ —— 群体7灾厄+格挡，兼顾攻防
  • 死亡使者+ ★★★★ —— 群体26灾厄+虚弱，铺场利器
  • 恐惧散布者 ★★★★ —— 施加灾厄时获得格挡，攻防一体

灾厄结算/爆发：
  • 末日降临+ ★★★★★ —— 37点AOE直伤，唯一能直接结算灾厄的卡，必拿
  • 湮灭 ★★★★★ —— 灾厄层数翻倍，消耗，爆发核心
  • 大限将至 ★★★★ —— 造成等同于目标灾厄层数的伤害，高层数收割

防御核心：

  • 拖延 ★★★★ —— 0费13格挡，前期过渡神卡

运转辅助：
  • 神经涌动 ★★★★ —— 0费过牌+能量，配合死亡之门叠甲
  • 领会 ★★★★ —— 虚无+抽4张，快速找关键牌
  • 降灵 ★★★★ —— 战斗中烧掉非关键牌，精简卡组
```

```
灾厄施加：

### 💡 通用攻略要点

#### 三大核心原则

```
1. 精简卡组
   → 保证关键牌上手率，不要无脑接牌
   → 使用降灵/雕琢打击持续烧掉不必要的牌

2. 攻防平衡
   → 当前版本生存标准：每回合造成100点伤害，获得40点格挡
   → 在高效攻击下，防御压力自然减小

3. 灵活转型
   → 不要开局就锁死单一流派
   → 根据获得的卡牌和遗物动态调整路线
```

#### 常见误区

> ❌ **不要**：以为奥斯提的伤害吃力量加成（它不吃）
> ❌ **不要**：没有稳定施加灾厄手段时无脑拿死亡之门
> ❌ **不要**：灾厄流只依靠灾厄清怪，前期必须有直伤过渡
> ✅ **要**：根据地图决策，有精英怪多时优先灾厄流，路线长时考虑无限循环流
