## 一、终端基础指令

| 指令 | 说明 |
|------|------|
| `Moons` | 选择飞船目的地 |
| `Store` | 打开商店购买装备 |
| `Bestiary` | 打开生物图鉴，查询已发现的生物信息 |
| `Storage` | 打开储藏室，可存取物品和道具 |
| `Other` | 展开其他命令，使用游戏内的功能 |

### 快捷键

| 快捷键 | 说明 | 来源 MOD |
|--------|------|----------|
| `Ctrl+J` | 堵塞所有炮塔和地雷（Jamming） | TerminalCommander |
| `Ctrl+D` | 打开/关闭所有设施门 | TerminalCommander |
| `Ctrl+E` | 将所有玩家紧急传送回飞船 | TerminalCommander |
| `Ctrl+W` | 传送（正向 Teleport） | TerminalCommander |
| `Ctrl+I` | 逆向传送（Inverse Teleport） | TerminalCommander |
| `Ctrl+S` | 切换监视器跟踪工作中的队友 | TerminalCommander |
| `Ctrl+T` | 开始发送广播（Start Transmission） | TerminalCommander |
| `Ctrl+M` | 开关监视器电源 | TerminalCommander |

> 以上快捷键默认字母可在 `Tronald.TerminalCommander.cfg` 中自定义。
