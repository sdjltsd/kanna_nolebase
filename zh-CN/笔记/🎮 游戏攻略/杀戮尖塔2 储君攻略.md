### 👑 角色概览

> **储君（The Sovereign）** 是《杀戮尖塔2》中的隐藏职业，以"双资源"作为核心玩法——除了常规能量外，还独立管理名为 **辉星（Starbeam）** 的次要资源，高傲地指挥星辉力量征服尖塔。

| 属性 | 数值 |
|------|------|
| 初始生命值 | **75 HP** |
| 初始能量 | 3 |
| 难度定位 | ⭐⭐⭐⭐ 中高复杂度 |
| 解锁条件 | 使用铁甲战士（Ironclad）通关一次 |
| 初始遗物 | **天赋君权** —— 每场战斗开始时获得 3 点辉星 |
| 初始卡组 | 打击 ×4、防御 ×4、陨星 ×1、崇拜 ×1 |

> 💡 **角色定位**：中等生命值的双资源型角色，需要同时管理能量与辉星两套资源。上手门槛中等，成型后"星星流"数值极强，爆发惊人。

### ⚙️ 核心机制详解

#### 1. ⭐ 辉星（Starbeam）—— 次要资源系统

辉星是储君独有的第二种资源，是整个角色运转的核心。

```
核心特性：
✦ 持久性：与每回合重置的能量不同，辉星可积累并跨回合保留
✦ 上限：辉星上限通常为10颗
✦ 获取：通过特定卡牌（如崇拜、辉光、隐秘藏品等）获取
✦ 消耗：许多强力卡需消耗辉星才能打出，或以辉星数量决定伤害
✦ 不自动回复：每回合不会自动产生辉星，须靠卡牌主动生成
```

> ⚠️ **重要**：辉星是储君最宝贵的资源，**不要浪费**。打出消耗辉星的强力卡时，确保已积累足够数量。

#### 2. 🌟 陨星（Meteor）—— 初始核心攻击

陨星是储君初始卡组自带的核心输出手段。

```
核心规则：
✦ 0费，消耗所有辉星，每星造成4点伤害（升级后5点）
✦ 10颗辉星时可打出40-50点伤害
✦ 升级版"流星雨"可对所有敌人造成群体伤害
✦ 是"星星流"体系的最终输出终端
```

#### 3. ⚔️ 锻造（Forging）—— 永久强化机制

锻造是储君的另一大特色，允许永久提升卡牌伤害。

```
核心规则：
✦ 带有"锻造"关键词的卡牌，可永久提升"君主之刃"的伤害
✦ 锻造层数累积后，每次打出君主之刃伤害都更高
✦ 是"大剑流"的核心成长机制
✦ 配合征服者（伤害翻倍）可打出毁灭性爆发
```

### 🎯 四大流派构筑

#### ⭐ 流派一：星星流（首选推荐）

> **核心思路**：通过大量获取辉星，利用消耗辉星的强力卡打出高额爆发伤害。容错率高，数值成型后堪称"崩坏"，是新手冲层最稳健的选择。

**必要组件：**

```
辉星生成核心：
  • 辉光（Glow）★★★★★ —— 加2星+抽2张，假卡
  • 隐秘藏品（Secret Cache）★★★★★ —— 单卡叠4辉星，破坏平衡
  • 明耀打击（Radiant Strike）★★★★ —— 每回合稳定+2辉星
  • 碰撞轨迹★★★★ —— 0费过渡兼加星

输出核心：
  • 陨星（Meteor）★★★★★ —— 每辉星4-5伤，零费高伤
  • 辐射（Radiation）★★★★ —— 循环体系主力，配活力成指数成长
  • 伽马爆破★★★★ —— 高额输出，附带双增益
  • 流星雨（升级陨星）★★★★★ —— 群体版陨星，清场神器

防御核心：
  • 倒映（Reflect）★★★★★ —— 消耗3辉星，17格挡+反弹
  • 群星之子（Child of Stars）★★★★ —— 攻防一体启动
  • 宇宙冷漠★★★★ —— 后期救命防御

续航辅助：
  • 创世纪（Genesis）★★★★ —— 长线续航基石
  • 类星体（Quasar）★★★★★ —— 持续印出好牌
  • 微光（Glimmer）★★★★ —— 缺牌时过牌续航
```

**推荐卡组（约18-22张）：**

```
辉星生成：
  ×1-2 辉光（必拿，升级优先）
  ×1-2 隐秘藏品
  ×1-2 明耀打击
  ×1-2 碰撞轨迹

输出核心：
  ×1-2 陨星（升级优先→流星雨）
  ×1-2 辐射
  ×0-1 伽马爆破

防御核心：
  ×1-2 倒映（必拿）
  ×1 群星之子（必拿）
  ×1 宇宙冷漠

续航辅助：
  ×1 创世纪（升级优先）
  ×1 类星体
  ×0-1 微光
  ×1 光子切割（升级优先）
```

**运转模式：**

```
标准模式：辉星积累 → 爆发循环
  1. 前期用崇拜+明耀打击维持辉星
  2. 找到辉光/隐秘藏品，快速叠到8-10颗辉星
  3. 打出陨星/流星雨造成40-50点高额伤害
  4. 用倒映等防御卡反弹小怪攻击
  5. 进入中后期，辐射+活力buff实现指数成长
  
  ⚠️ 注意辉星上限10颗，超量会浪费！合理规划消耗时机
  
进阶模式：倒映反弹流
  1. 用霸权跨回合积累费用
  2. 打出倒映（消耗3辉星）获得17格挡+伤害反弹
  3. 配合群星之子持续补充辉星
  4. 让敌人在反弹中自我消耗，实现防御输出一体化
  
  ⚠️ 针对高频小攻击的精英怪效果拔群
```

**升级优先级：**

```
第一优先：
  ✦ 陨星 → 每星伤害4→5，10星即50点伤害
  ✦ 辉光 → 解锁抽牌效果，双重价值飞跃
  ✦ 光子切割 → 1费打13+抽2回1，飞跃式提升

第二优先：
  ✦ 创世纪 → 费用降低，续航更流畅
  ✦ 倒映 → 格挡数值提升
  ✦ 辐射 → 伤害与辉星联动增强

第三优先：
  ✦ 微光 → 过牌更流畅
  ✦ 明耀打击 → 辉星产量提升
```

#### ⚔️ 流派二：大剑流（慢热爆发型）

> **核心思路**：通过锻造机制持续强化"君主之刃"的伤害，配合虚空形态降费和征召上前回收，在一回合内反复打出这把大剑，造成毁灭性的单点爆发。成型后体验极爽，但容错率低、成型慢，不建议强行构建。

> ⚠️ **注意**：大剑流非常依赖发牌顺序，前期弱势。必须先确认能够稳定找到核心组件，再考虑走这条路线。

**必要组件：**

```
绝对核心（缺一不可）：
  • 君主之刃（Sovereign's Blade）★★★★★ —— 2费保留，锻造的伤害载体
  • 虚空形态（Void Form）★★★★★ —— 使君主之刃本回合变为0费
  • 征召上前（Summon Forward）★★★★★ —— 从弃牌堆/消耗堆回收君主之刃
  • 抉择/锻打成型★★★★★ —— 高辉星消耗，大量锻造层数
  • 征服者（Conqueror）★★★★★ —— 君主之刃伤害翻倍，斩杀核心

锻造辅助：
  • 铸墙（Forge Wall）★★★★ —— 同时提供格挡和锻造，攻防兼顾
  • 宇宙冷漠（Cosmic Indifference）★★★★ —— 君主之刃下回合自动返手
  • 淬炼刀刃（Tempered Blade）★★★ —— 低费附带小伤的锻造牌
  • 铸剑者/剑动力学★★★ —— 低费高效锻造过渡

前期过渡：
  • 辉光、光子切割 —— 前期过牌+产辉星，保证活到成型
  • 倒映、群星之子 —— 前期防御生存
```

**推荐卡组（约20-22张）：**

```
核心组件：
  ×1 君主之刃（必拿）
  ×1-2 虚空形态（必拿）
  ×1-2 征召上前（必拿）
  ×1 征服者（必拿）
  ×1 抉择/锻打成型

锻造辅助：
  ×1 铸墙
  ×1 宇宙冷漠
  ×1-2 淬炼刀刃/铸剑者

前期过渡（后期可考虑剔除）：
  ×1-2 辉光
  ×1-2 光子切割（升级优先）
  ×1-2 倒映
  ×1 群星之子
```

**三阶段作战思路：**

```
第一阶段（1层）：生存过渡
  → 不主动抓锻造牌，优先辉光/光子切割过牌产星
  → 倒映/群星之子维持防御
  → 避免不必要的精英战

第二阶段（2层）：构筑成型
  → 集齐虚空形态+征召上前后开始发力
  → 每回合用辉星产锻造层数
  → 铸墙等兼顾防御与成长
  → 留着高层数君主之刃打精英/BOSS

第三阶段（3层+）：终极爆发
  经典连招（成型后）：
  1. 开启虚空形态（君主之刃→0费）
  2. 打出君主之刃
  3. 征召上前拉回手牌
  4. 抉择/锻打成型大幅提升伤害
  5. 打出征服者（伤害翻倍）
  6. 再次打出君主之刃 → 征召 → 循环斩杀
  
  ✦ 成型标志：君主之刃基础伤害超过50点
```

**升级优先级：**

```
第一优先：
  ✦ 虚空形态 → 更低激活成本
  ✦ 征召上前 → 回收效率提升
  ✦ 征服者 → 爆发翻倍效率提升

第二优先：
  ✦ 铸墙 → 格挡+锻造效率双提升
  ✦ 淬炼刀刃 → 降费，存费配合更灵活
  ✦ 光子切割 → 前期继续过渡
```

**必要组件：**

```
绝对核心：
  • 类星体（Quasar）★★★★★ —— 印卡流核心，三选一产出好牌

功能配合：

  • 所向无敌★★★★ —— 优质防御，配合过牌
  • 粒子墙★★★ —— 配合霸权稳定获得大量格挡
```

```
第一优先：

  ✦ 霸权 → 跨回合积累更多费用
  ✦ 胜券在王 → 移除消耗，可重复打出
```

#### 🎲 流派四：异蛇赌狗流（特殊开局）

### 💡 通用攻略要点

#### 三大核心原则

```
1. 生存优先
   → 第一层切勿盲目追求输出，优先抓"倒映"等防御牌
   → 利用双资源（辉星）打防守反击，活下来才有输出
   → 倒映反弹小怪是储君特有的生存手段

2. 精简卡组
   → 进入第二层后开始极限压缩牌库
   → 剔除前期过渡用的白卡"废件"（如崇拜、太阳打击）
   → 确保每回合能稳定打出核心牌循环
   → 目标：最终卡组 20 张左右

3. 定向抓牌
   → 明确构筑方向（通常优先走星星流）后只抓符合方向的卡
   → 不要"既要又要"，避免卡组臃肿卡手
   → 克制抓牌欲望，宁缺勿滥
```

#### 常见误区

> ❌ **不要**：没有稳定辉星来源时盲目拿辐射等消耗辉星的卡
> ❌ **不要**：被铸剑体系的高伤害数字迷惑，当前版本铸剑整体弱势
> ❌ **不要**：卡组超过25张后还继续无脑拿牌
> ❌ **不要**：前期把辉星全部浪费在小战斗，大Boss前应积累满辉星
> ✅ **要**：优先升级陨星和辉光，这是最划算的升级
> ✅ **要**：有倒映时主动面对高频攻击精英，用反弹弹死对方
> ✅ **要**：根据地图路线灵活决策，长路线适合星星流，短路线可赌大剑流

#### 路线选择思路

```
精英路线多时：
  → 优先星星流，陨星伤害高，精英输出充足
  → 倒映反弹可以"白嫖"精英的连续攻击

火焰路线多时：
  → 优先拿防御牌（倒映、群星之子）
  → 保留辉星不要浪费在过渡战斗

商店路线：
  → 优先升级陨星和辉光
  → 有多余金币可移除崇拜/太阳打击等前期废牌
```
